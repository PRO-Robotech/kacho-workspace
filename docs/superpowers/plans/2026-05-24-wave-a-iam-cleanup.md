# Wave A — kacho-iam Comment + KAC + Phase Cleanup — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Strip every `Phase \d+[a-z]?`, `E[0-9]`, `W[0-9]\.[0-9]` and historical `KAC-N` reference from `kacho-iam` source artefacts (production + test + newman + migrations comments), rewording each comment to preserve its «WHY» while dropping its «WHEN».

**Architecture:** Pure mechanical-with-judgment refactor. Zero behavioural change. Per-folder batched commits with go-build + go-test + newman-e2e regression gates between commits. No file renames (Wave B). No structural reshaping (Wave D). No migration file renaming (Wave M).

**Tech Stack:** `git mv` for any test-function rename impact (none expected — tests stay in same files); `gofmt` post-edit; `goimports`; `go vet`; `go test`; `grep`-based acceptance audit script (`scripts/wave-a-audit.sh` created in Task 1).

**Tracking**: KAC-195 (sub-ticket of KAC-194 epic). Branch: `KAC-195-wave-a-cleanup` in `kacho-iam`. Worktree: `/tmp/wt-KAC-195-iam`.

**Time budget**: ~3 working days end-to-end (12-15 commits × 30-60 min each).

```bash
#!/usr/bin/env bash
# scripts/wave-a-audit.sh — count Phase + KAC + W refs in kacho-iam (Wave A acceptance gate).
# Excludes: verifies-KAC-N annotations (workspace §13 finding pattern), migration filenames (Wave M), commits.

set -euo pipefail
cd "$(dirname "$0")/.."

print_kind() {
  local label="$1"; shift
  local count
  count=$(grep -rln "$@" --include='*.go' --include='*.py' --include='*.sql' internal/ cmd/ tests/newman/cases/ 2>/dev/null | wc -l)
  printf '%-40s %5d\n' "$label" "$count"
}

print_kind "Phase \\d (markers in code)"                  -E 'Phase [0-9]'
print_kind "\\bE[0-9]\\b (markers)"                       -E '\bE[0-9]\b'
print_kind "\\bW[0-9]\\.[0-9]\\b (Wave markers)"          -E '\bW[0-9]\.[0-9]\b'
print_kind "KAC-N total"                                  -E 'KAC-[0-9]'
print_kind "KAC-N legitimate (// verifies / # verifies)"  -E '(//|#) *verifies.*KAC-[0-9]'

echo
echo "Wave A goal: first 3 metrics ≤ 0; last (verifies-KAC) ≤ 10."
```
