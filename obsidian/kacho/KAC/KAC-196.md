## Что и зачем

Реализация InternalClusterService с RPC Get / GrantAdmin / RevokeAdmin / ListAdmins.
Даёт admin-tooling ability управлять кластерными администраторами (system_admin FGA tuple)
через internal API (:9091). Acceptance §6.00–§6.12.

Гарантии: D-2 (USER only), D-4 (idempotent Grant), D-5 (no self-revoke),
D-6 (last-admin lock-out), D-9 (user must exist), D-12 (revoke only active).
